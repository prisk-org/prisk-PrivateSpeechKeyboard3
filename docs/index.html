Architecture

┌─────────────────────┐    prisk://start-recording    ┌──────────────────┐
│   PriskKeyboard     │ ──────────────────────────▶ │    PriskApp      │
│  (Keyboard Ext.)    │                             │  (STT Host)      │
│                     │    Darwin Notification       │                  │
│  • QWERTY UI        │ ◀────────────────────────── │  • WhisperKit    │
│  • Mic button       │    App Group UserDefaults    │  • SFSpeech      │
│  • Suggestion bar   │                             │  • SpeechAnalyzer│
└─────────────────────┘                             └──────────────────┘

Keyboard extensions have a ~70MB memory limit. Prisk offloads STT to the main app (which can use 200MB+ for WhisperKit) and communicates via App Group + Darwin Notifications.
